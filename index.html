Application Overview

SnapShift is a modern web app that allows users to easily convert images between different formats (JPG, PNG, BMP, PDF) with a clean, drag-and-drop interface. Users can upload images, select the output format, and instantly download the converted fileâ€”no signups or hassle. The app is responsive, includes dark mode support, and provides real-time preview and success feedback during conversions.

Technology Overview

  • Languages: Python, HTML, CSS, JavaScript
  • Libraries: Flask (backend), Pillow (image processing)
  • Platform: Desktop web browser
  • Hosting: GitHub Pages (frontend), Render or PythonAnywhere (backend)

Feature List (MVP)

  • Drag-and-drop image upload zone
  • Automatic format detection
  • Selectable output formats (JPG, PNG, BMP, PDF)
  • Instant image conversion and download
  • Loading spinner and success message after conversion
  • Mobile-friendly responsive design

Future Feature List

  • Batch image conversion (multiple files)
  • Image compression and resizing tools
  • Cloud storage integration (Google Drive, Dropbox)
  • Enhanced preview options (zoom, rotate)
  • Dedicated mobile app version

Basic Technical Features

  • Python Flask backend server
  • Secure file upload and handling
  • Image processing using Pillow (Python Imaging Library)
  • Frontend with HTML, TailwindCSS, and Vanilla JavaScript
  • Hosting using GitHub Pages (frontend) and Render (backend)

Advanced Technical Features

  • Real-time UI updates with JavaScript transitions
  • Dark Mode toggle functionality
  • Loading spinner during processing
