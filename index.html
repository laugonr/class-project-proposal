Application Overview

SnapShift is a user-friendly web app that makes image format conversion quick and hassle-free. Upload your file, select a new format (PNG, JPG, or PDF), and convert it instantlyâ€”no signups, ads, or confusion.

Technology Overview

  • Languages: Python, HTML, CSS, JavaScript
  • Libraries: Flask (backend), Pillow (image processing)
  • Platform: Desktop web browser
  • Hosting: GitHub Pages (frontend), Render or PythonAnywhere (backend)

Feature List (MVP)

  • Drag-and-drop image upload
  • Selectable output formats (JPG, PNG, PDF)
  • Instant image conversion
  • Downloadable converted file

Future Feature List

  • Batch conversion
  • Image compression/resizing
  • Dark mode interface
  • Image preview before Download

Basic Technical Features

  • Flask-based backend
  • Secure file upload handling
  • Image format conversion using Pillow
  • Frontend built with HTML/CSS/JavaScript

Advanced Technical Features

  • Batch upload(Multiple Files)
